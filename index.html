Information Visualization

This website is a showcase of the work we have done throughout CS725, Information Visualization. In this course, we are learning how to create effective information visualizations as well as how to use tools such as D3 and tableau.

The work on this site is a collaboration between Kayla Henneman and Swaraj Wankhade.
